# Spell
Spell Script Language - Programming with Harry Potter spells

How to Use

Use spellc to convert a Spell script to the C Language:

./spellc test.spell

Produces a file named code.c

Use spellrun to run a Spell script in the CLI:

./spellrun test.spell
